# 📊 Marketing Campaign Analysis Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Marketing Campaign Analysis Dashboard built using Power BI. The dashboard helps analyze marketing performance by comparing campaign spending, revenue generation, return on investment (ROI), regional performance, and campaign types. It enables businesses to evaluate campaign effectiveness and optimize marketing strategies.

---

## 🎯 Objectives

- Analyze marketing campaign performance.
- Compare campaign spending with generated revenue.
- Measure ROI across different campaigns.
- Identify the best-performing marketing campaign.
- Evaluate regional and campaign-type performance.

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV Dataset
- Data Modeling

---

## 📂 Dataset

The dataset contains marketing campaign information, including:

- Campaign Name
- Campaign Type
- Region
- Marketing Spend
- Revenue Generated
- Return on Investment (ROI)
- Campaign Performance Metrics

---

## 🧹 Data Preparation

The dataset was cleaned and transformed using **Power Query** by:

- Removing duplicate records
- Handling missing values
- Correcting data types
- Standardizing campaign names
- Validating numerical values
- Creating calculated columns for analysis

## 📈 Key Insights

### Revenue Analysis

- Total revenue generated reached **42.54M**, exceeding the overall marketing spend.

### Campaign Performance

- **Influencer Marketing** delivered the highest ROI among all campaigns.
- Social Media and Search Engine campaigns also performed strongly.

### Regional Analysis

- Africa generated the highest combined revenue and marketing spend.
- Performance varied across regions, indicating opportunities for budget optimization.

### Campaign Type

- Digital campaigns accounted for approximately **62.5%** of total campaigns.
- Traditional campaigns represented **37.5%**.

### ROI Analysis

- The average ROI across all campaigns is **0.68**.
- Several campaigns outperformed the average, suggesting successful investment strategies.

---

## 💼 Business Recommendations

- Increase investment in high-performing campaigns such as Influencer Marketing.
- Reallocate budget toward regions delivering stronger ROI.
- Expand successful digital marketing initiatives.
- Regularly monitor campaign ROI to improve marketing efficiency.
- Optimize underperforming campaigns through targeted strategy adjustments.
